# M6: Intel E1000 网卡驱动

> QEMU 默认网卡。register-based 操作，适合作为第一个网卡驱动。
> 为 F7 网络协议栈提供数据链路层。

## 目标

实现 E1000 驱动，支持以太网帧收发。

## 任务清单

### T1: E1000 初始化

**文件**: `kernel/drivers/net/e1000.hpp`, `kernel/drivers/net/e1000.cpp`

```cpp
namespace cinux::drivers::net {

class E1000 {
public:
    void init(const pci::PCIDevice& pci_dev);

    // 帧收发
    bool send(const void* data, size_t len);
    int32_t receive(void* buf, size_t buf_size);

    // MAC 地址
    void get_mac(uint8_t mac[6]);

    // 状态
    bool link_up();
    uint32_t supported_speed();

private:
    volatile uint32_t* regs_;  // MMIO 基地址
    DmaBuffer rx_ring_;        // 接收描述符环
    DmaBuffer tx_ring_;        // 发送描述符环
    DmaBuffer* rx_buffers_;    // 接收缓冲区数组
    uint32_t rx_tail_;
    uint32_t tx_tail_;
    uint8_t  mac_[6];
};

} // namespace cinux::drivers::net
```

**E1000 寄存器** (部分):
| 偏移 | 名称 | 用途 |
|------|------|------|
| 0x0000 | CTRL | 设备控制 |
| 0x0008 | STATUS | 设备状态（link speed） |
| 0x1000 | RCTL | 接收控制 |
| 0x2800 | TCTL | 发送控制 |
| 0x5400 | RDBAL/RDBAH | RX 描述符环基址 |
| 0x5800 | TDBAL/TDBAH | TX 描述符环基址 |
| 0x5400+MAC | RAL/RAH | MAC 地址过滤 |

- [ ] PCI 发现（vendor=0x8086, device=0x100E）
- [ ] MMIO 映射
- [ ] 读取 EEPROM MAC 地址
- [ ] 配置接收/发送描述符环（各 256 entries）
- [ ] 分配接收缓冲区（DMA Pool）
- [ ] 设置中断掩码

### T2: 以太网帧收发

**RX 描述符格式** (Legacy):
```
[63:0]   Address    — 缓冲区物理地址
[79:64]  Length     — 接收到的帧长度
[80]     Done       — DD bit（描述符完成）
[81]     End of Packet
```

**TX 描述符格式** (Legacy):
```
[63:0]   Address    — 缓冲区物理地址
[79:64]  Length     — 帧长度
[80]     DD bit
[81]     EOP (End of Packet)
[82]     IFCS (Insert FCS)
[83]     RS (Report Status)
```

- [ ] send() — 填充 TX 描述符 + DMA 缓冲区
- [ ] receive() — 检查 RX 描述符 DD bit + 拷贝数据
- [ ] 环形缓冲区 wrap-around 处理
- [ ] 中断驱动接收（RXDMT0 中断）

### T3: 网络设备抽象

**文件**: `kernel/drivers/net/net_device.hpp`

```cpp
class INetDevice {
public:
    virtual ~INetDevice() = default;

    virtual bool send(const void* data, size_t len) = 0;
    virtual int32_t receive(void* buf, size_t buf_size) = 0;
    virtual void get_mac(uint8_t mac[6]) = 0;
    virtual bool link_up() = 0;

    // MTU
    virtual uint32_t mtu() const { return 1500; }
};
```
