# M1: FAT32 文件系统

> FAT12/16/32 统一实现。支持 UEFI ESP 分区和 U 盘互操作。
> 只读 + 基础写入。

## 任务清单

### T1: FAT 卷结构

**文件**: `kernel/fs/fat.hpp`, `kernel/fs/fat.cpp`

```cpp
class FatFS : public FileSystem {
public:
    bool mount() override;
    Inode* lookup(const char* path) override;

private:
    // BPB (BIOS Parameter Block)
    struct BootSector {
        uint8_t  jmp[3];
        char     oem_name[8];
        uint16_t bytes_per_sector;    // 通常 512
        uint8_t  sectors_per_cluster;
        uint16_t reserved_sectors;
        uint8_t  num_fats;
        uint16_t root_entry_count;    // FAT12/16 only
        uint16_t total_sectors_16;
        uint8_t  media_type;
        uint16_t sectors_per_fat_16;
        // ... extended fields for FAT32 ...
        uint32_t total_sectors_32;
        uint32_t sectors_per_fat_32;
        uint32_t root_cluster;        // FAT32 根目录起始簇
    };

    IBlockDevice* dev_;
    uint32_t fat_start_;
    uint32_t data_start_;
    uint32_t root_cluster_;
    uint32_t cluster_count_;
    uint8_t  fat_type_;  // 12, 16, or 32
};
```
